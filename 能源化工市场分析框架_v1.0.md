### 3.4 各品种关注重点

#### HH（美国天然气）
- **核心关注**：EIA库存、产量、天气（HDD/CDD）、LNG出口
- **季节性特征**：冬季（11-3月）和夏季（6-8月）为需求高峰
- **关键价位**：$2/MMBtu（成本支撑）、$5/MMBtu（需求抑制）

#### TTF（欧洲天然气）
- **核心关注**：欧洲库存水平、俄罗斯管道气、LNG进口、天气
- **地缘因素**：俄乌冲突、北溪管道、制裁政策
- **关键价位**：€30/MWh（历史常态）、€100/MWh（危机水平）

#### JKM（亚洲LNG）
- **核心关注**：日韩库存、中国需求、全球LNG供应、油价挂钩
- **季节性特征**：冬季（12-2月）为需求高峰
- **定价特点**：与油价挂钩+现货溢价

| 汽油裂解价差 | 汽油价格-原油价格 | 汽油利润 |
| 柴油裂解价差 | 柴油价格-原油价格 | 柴油利润 |
| 汽柴价差 | 汽油价格-柴油价格 | 品种相对强弱 |

### 7.2 综合评分计算

#### 7.2.1 评分公式

```
综合评分 = Σ(维度得分 × 维度权重)

其中：
- 各维度得分 = Σ(子指标得分 × 子指标权重)
- 得分范围：-5（强烈看空）到 +5（强烈看多）
- 0分表示中性
```

#### 7.2.2 评分等级

| 综合评分 | 评级 | 建议 |
|---------|-----|-----|
| +4~+5 | 强烈看多 | 积极做多 |
| +2~+4 | 看多 | 偏多操作 |
| 0~+2 | 轻度看多 | 谨慎偏多 |
| -2~0 | 轻度看空 | 谨慎偏空 |
| -4~-2 | 看空 | 偏空操作 |
| -5~-4 | 强烈看空 | 积极做空 |
